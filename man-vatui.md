# 一、引言
本使用手册旨在帮助开发者了解和使用基于 `VaTui` 相关类实现的终端用户界面（TUI）操作功能库。该库涵盖光标控制、颜色设置、终端属性操作以及字符编码相关判断等功能，有助于打造丰富友好的终端应用交互界面。

# 二、整体结构概述
整个功能库围绕 `VaTui` 类构建，整合了多个功能子模块，包括 `VaCursor`（光标操作）、`VaColor`（颜色控制）、`VaTerm`（终端属性与屏幕操作）、`VaSystem`（系统信息获取）、`VaUtf`（字符编码判断）。创建 `VaTui` 类实例后，可调用各子模块功能方法操作终端界面。

# 三、VaTui 类使用说明
`VaTui` 类的构造函数自动初始化各子模块实例，析构函数在对象生命周期结束时释放子模块实例占用的内存资源，避免内存泄漏。示例代码如下：

```cpp
#include "VaTui.hpp" 

int main() {
    VaTui myTui; // 创建实例，初始化各子模块
    return 0;
}
```

# 四、VaColor 类功能使用
## 4.1 颜色设置
### 4 位颜色模式设置
使用 `SetColor4bit` 方法，通过 `color4bit` 命名空间下的枚举值指定前景色和背景色。返回值为空，该方法直接设置颜色效果。

### 16 位颜色模式设置
`SetColor256` 方法接收对应颜色代码来设置文本和背景颜色，无返回值，作用是改变颜色显示。

### RGB 颜色模式设置（需终端支持）
`set_background_color_RGB` 和 `set_front_color_RGB` 方法用于设置背景和前景全彩色（RGB 模式），需传入 `R`、`G`、`B`（0 - 255）三个分量值，无返回值，用于调整颜色呈现效果。

## 4.2 文本效果控制
`SetEffect` 方法结合 `AnsiEffect` 枚举设置文本显示效果（如加粗、斜体等），传入效果枚举值和布尔值（表示启用与否），无返回值，直接改变文本显示样式。

## 4.3 颜色效果重置
调用 `_ColorEffectReset` 方法（通过 `Color` 成员对应接口）可将文本颜色和效果恢复默认，无返回值，用于还原初始显示状态。

# 五、VaCursor 类功能使用
## 5.1 光标位置移动
### 移动到指定位置
`CursorMoveTo` 方法传入目标位置的行（`h`）和列（`w`）坐标，将光标移动到指定处，无返回值，用于精准定位光标。

### 按方向和距离移动
`CursorMove` 方法接收移动方向枚举值（如 `CUR_LEFT`、`CUR_RIGHT` 等）及移动距离，按指定方向移动光标，无返回值，实现光标相对移动。

## 5.2 光标显示控制
### 隐藏光标
`CursorHide` 方法用于隐藏光标，无返回值，适用于特定界面展示需求。

### 显示光标
`CurShow` 方法重新显示被隐藏的光标，无返回值，恢复正常光标可见性。

## 5.3 光标位置重置
`CursorReset` 方法能将光标重置到默认位置（通常是终端左上角，即第一行第一列），无返回值，方便还原光标初始位置。

# 六、VaTerm 类功能使用
## 6.1 终端屏幕操作
### 清除整个屏幕
`Clear` 方法清除终端屏幕所有内容，无返回值，快速清空屏幕显示。

### 清除从光标位置到行尾内容
`ClearLine` 方法清除光标所在位置到当前行末尾内容，无返回值，用于局部屏幕清理。

## 6.2 终端属性控制
### 获取终端尺寸
`getTerminalSize` 方法获取当前终端窗口行数和列数，通过参数返回获取到的行数和列数，用于知晓终端显示范围。

### 控制终端输入回显
`enableEcho` 和 `disableEcho` 方法分别用于启用和禁用终端输入回显，无返回值，以控制输入内容是否显示。

### 控制终端输入缓冲机制
`enableConsoleBuffering` 和 `disableConsoleBuffering` 方法用于控制终端输入缓冲行为，无返回值，影响输入获取方式。

### 保存与恢复光标位置
`saveCursorPosition` 方法保存当前光标位置，无返回值；`restoreCursorPosition` 方法恢复之前保存的光标位置，无返回值，便于界面操作后还原光标位置。

# 七、VaSystem 类功能使用
## 7.1 获取用户名
`getUserName` 方法通过系统相关函数获取当前系统用户名，返回 `std::string` 类型字符串，表示用户名，若获取失败返回空字符串。

## 7.2 获取当前时间
`getCurrentTime` 方法获取系统当前时间，并格式化为 `%Y-%m-%d %H:%M:%S` 格式的字符串返回，返回值类型为 `std::string`。

## 7.3 获取环境变量
`getRunningEnvironment` 方法接收环境变量名（如 `"PATH"`），通过 `std::getenv` 函数查找并返回对应环境变量值，返回值为 `std::string` 类型，若变量不存在或获取失败则返回空字符串。

## 7.4 获取设备名称
`getDeviceName` 方法调用 `uname` 函数获取 `utsname` 结构体中的机器名近似作为设备名称，返回 `std::string` 类型字符串，若调用失败返回空字符串。

## 7.5 获取主机名
`getHostName` 方法借助 `gethostname` 函数获取主机名并转换为 `std::string` 类型返回，若获取失败返回空字符串。

## 7.6 获取运行目录
`getRunningDirectory` 方法使用 `getcwd` 函数获取程序当前运行目录路径，返回 `std::string` 类型字符串，若调用失败返回空字符串。

# 八、VaUtf 类功能使用
## 8.1 判断字符编码相关
### 判断是否为 ASCII 字符
`isAscii` 方法传入一个字符，判断是否为 `ASCII` 字符，返回布尔值，`true` 表示是，`false` 表示否。

### 判断是否为 UTF-8 编码字符
`isUtf8Char` 方法传入字节序列及其长度，判断是否为合法 `UTF-8` 编码字符，返回布尔值，代表判断结果。

### 判断是否为 GBK 编码字符（简单判断）
`isGbkChar` 方法传入两个字节序列，依据简单字节范围判断是否可能为 `GBK` 编码字符，返回布尔值体现判断情况。
